== Description ==
Logito Cash On Delivery And Online Payment

مشتریان شرکت لجیتو میتوانند حساب لجیتوی خود را به این افزونه متصل نمایند و محصولات خود را از طریق روش های ارسال لجیتو و درگاه آن برروی فروشگاه ساز ووکامرس بفروشند.


== Installation ==
1. Upload `wp-logito.zip` to the \'/wp-content/plugins/\' and extract it
2. Activate the plugin through the \'Plugins\' menu in WordPress
3.Go to settings --> configure logito plugin and enter your logito account details.
4. Go to woocommerce configuration >> checkout >> cash on delivery >> fill cash on delivery checkbox and select new shipping and payment methods from the bottom box.
5. Go to woocommerce configuration >> shipping >> go to each new shipping and payment methods section and activate it.
